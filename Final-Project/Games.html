Rhythm City Runner

Rhythm City Runner is a game designed by Rick Starkweather and me for the Creative Technology Game Development Final. It is an autorunner game where the entire game changes depending on what song is playing. The BPM changes with the speed of the song, the level is procedurally generator with the song, and the enemy placements are depentent on where in the song it is. It is currently just an Alpha Build made for class, but I might one day go back and work on it more.

Paradoxical Puzzle Room

Paradoxical Puzzle Room is a prototype point and click adventure desgined by Logan McGrew and I. It is a simple escape room puzzle game with mystery elements weaved into the plot. It was made for the final exam of one of my classes, but this is a type of game I would like to revisit and make into a fully fledged game.
